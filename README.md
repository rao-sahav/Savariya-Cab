# Savariya Cab - Taxi Service Website

A complete, production-ready taxi service website for Savariya Cab based in Mathura, Uttar Pradesh, India.

## Tech Stack
- React + Vite
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features
- Fully responsive, mobile-first design
- Multiple pages (Home, About, Services, Fleet, Contact)
- WhatsApp booking integration
- Google Maps integration
- SEO-friendly structure
- Fast loading and optimized
